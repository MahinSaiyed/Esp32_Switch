🔌 ESP32 Dual Relay Control using Blynk (v1/v2)

This project allows you to control two relays (Relay 1 & Relay 2) using an ESP32 board and the Blynk IoT platform.
You can turn appliances ON/OFF from anywhere using the Blynk mobile app.

🚀 Features

Control 2 relays from Blynk (V1 & V2)

Real-time feedback to Blynk (V3 & V4)

Auto-OFF state on boot

Stable WiFi + Blynk connection

Clean, readable code with comments

🛠 Hardware Required

ESP32 Dev Board

2-channel Relay Module

Jumper Wires

Power supply (5V for relay)

🔌 Relay Pin Mapping (ESP32)
Relay Name	ESP32 Pin	Mode
Relay 1	GPIO 33	Output
Relay 2	GPIO 32	Output
📱 Blynk Virtual Pins
Virtual Pin	Function
V1	Control Relay 1
V2	Control Relay 2
V3	Feedback Relay 1
V4	Feedback Relay 2
